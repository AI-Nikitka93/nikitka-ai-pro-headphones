# Nikitka AI PRO

Premium fictional headphones portfolio site built as a static interactive product concept.

## What is inside

- Cinematic product hero with local optimized AVIF/WebP/JPG assets.
- Interactive product anatomy section with a local muted inline video.
- Live radio demo using public SomaFM streams.
- Simulated assistant and product showcase interactions.
- Truthful concept framing: this is a fictional portfolio product, not a real preorder page.

## Run locally

```bash
python -m http.server 4178
```

Then open `http://127.0.0.1:4178/`.

## Verification

The site was checked locally in browser on desktop and mobile before publication.
