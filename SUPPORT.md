# Support

Nikitka AI PRO is a portfolio demo, not a commercial product.

Use GitHub issues only for:

- live site rendering problems;
- broken links;
- documentation corrections;
- accessibility or responsive-layout issues;
- reproducible browser bugs.

Do not use issues for:

- requests to copy or reuse the site;
- commercial support for real headphones;
- vulnerability details;
- unrelated product promotion.

For permission, hiring, collaboration, or portfolio review questions, contact the repository owner through GitHub.
