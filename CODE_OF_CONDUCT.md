# Code of Conduct

This repository is maintained as a personal portfolio showcase.

## Expected Behavior

- Be respectful and specific.
- Keep feedback related to the website, documentation, accessibility, or portfolio presentation.
- Do not harass, spam, impersonate, threaten, or pressure the maintainer.
- Do not use this repository to request unauthorized reuse of protected code, assets, or brand materials.

## Enforcement

The maintainer may close issues, delete comments, block users, or limit interactions when discussion becomes abusive, spammy, unrelated, or unsafe.
