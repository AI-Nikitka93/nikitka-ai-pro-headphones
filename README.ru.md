# Nikitka AI PRO

Портфолио-сайт вымышленных премиальных наушников, собранный как AI-assisted one-hour concept.

[Сайт в интернете](https://nikitka-ai-pro-headphones.vercel.app/) | [README in English](README.md) | [Архитектура](docs/architecture.md) | [Аудит упаковки](docs/GITHUB_PACKAGING_AUDIT.md)

## Что Это

Nikitka AI PRO показывает, как можно быстро собрать премиальный продуктовый сайт с помощью ИИ: продумать визуальную подачу, собрать медиа, сделать интерактив, проверить страницу в браузере и довести до публичного деплоя.

Проект нужен для портфолио: он демонстрирует вкус, скорость, работу с ассетами, Three.js, видео, Web Audio API и честную подачу вымышленного продукта.

## Что Работает

- Кинематографичный hero для fictional consumer-tech продукта.
- Локальные оптимизированные ассеты: AVIF, WebP, JPG, PNG, MP4 и GLB.
- 3D-разбор кейса на Three.js: собранный кейс, крышка, внутренности, путь сигнала.
- Радиодемо через публичные потоки SomaFM и Web Audio API.
- Честная подача: наушники и характеристики вымышленные, но интерактив сайта реальный.
- Статический деплой на Vercel без backend, базы данных, авторизации и секретов.

## Локальный Запуск

Установка зависимостей не нужна.

```bash
python -m http.server 4178
```

Открыть:

```text
http://127.0.0.1:4178/
```

## Структура

```text
.
├── index.html                 # Разметка статической страницы
├── style.css                  # UI, адаптив, motion-стили
├── app.js                     # Интерактив, видео, радио
├── three-anatomy.js           # Three.js GLB viewer
├── assets/                    # Изображения, видео, постеры, GLB
├── tools/                     # Blender-генератор GLB-модели
├── docs/                      # Архитектура, аудит, история
└── .github/                   # Issue/PR templates и repo health files
```

## Проверка

Перед публикацией изменений полезно запускать:

```bash
node --check app.js
node --check three-anatomy.js
python -m py_compile tools/create_nikitka_product_model.py
git diff --check
```

Браузерная проверка должна покрывать desktop/mobile, блок `#product-3d`, видео, радио, консольные ошибки и горизонтальное переполнение.
