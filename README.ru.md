# Nikitka AI PRO

Премиальный статический сайт вымышленных наушников для портфолио.

Сайт в интернете: https://nikitka-ai-pro-headphones.vercel.app/

## Что работает

- Кинематографичный hero с локальными AVIF/WebP/JPG изображениями.
- Интерактивная анатомия продукта с локальным muted inline video.
- Радиодемо на публичных потоках SomaFM.
- Демо-ассистент и интерактивная витрина моделей.
- Честная подача: это вымышленный portfolio concept, а не реальный предзаказ.

## Локальный запуск

```bash
python -m http.server 4178
```

Открыть `http://127.0.0.1:4178/`.

## Проверка

Сайт проверен локально в браузере на desktop и mobile перед публикацией.
